# Halfmoon Organic Farm Website

Welcome to Halfmoon Organic - a small organic farm dedicated to growing fresh, sustainable produce for our local community.

## About This Website

This is a simple, professional website for Halfmoon Organic Farm, featuring:

- **Home Page**: Welcome message and farm highlights
- **About Page**: Farm story and sustainable farming practices
- **Products Page**: Seasonal organic produce offerings
- **Contact Page**: Contact information and ordering details

## How to View

Simply open `index.html` in your web browser to view the website locally.

To serve the website:
```bash
# Using Python 3
python3 -m http.server 8000

# Using PHP
php -S localhost:8000

# Using Node.js (with npx)
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## Customization

To customize this website for your farm:

1. Update contact information in `contact.html`
2. Modify product listings in `products.html` based on what you grow
3. Update the farm story in `about.html`
4. Adjust colors in `styles.css` (search for color values like `#2d5016`)
5. Add your own farm photos and logo
